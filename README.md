**Установить на локальной машине**

Клонируйте репозиторий командой:

```
git clone
```

Затем установите все зависимости командой(если есть node.js, если нет, то придется установить):

```
npm install
```

Затем запустите приложение командой в терминале:

```
npm run start
```
